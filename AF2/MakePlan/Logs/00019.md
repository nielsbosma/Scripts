In the excel inspector we'd like to add a screenshot <RANGE> or <SHEET> command.   What's the best nuget we can use to achieve this that don't have a copyleft license.

## Outcome

Created plan **393** — research plan for Excel Inspector screenshot command NuGet selection.

**Key finding:** Open-source Excel libraries (ClosedXML, EPPlus, NPOI, etc.) cannot render to images. Only commercial libraries have rendering engines.

**Top recommendations:**
- **Aspose.Cells** — best rendering, most mature, ~$1,175+
- **FreeSpire.XLS** — best free option with direct range-to-image API (3-page limit)
- **Syncfusion XlsIO** — free community license for small teams

**Plan file:** `393-General-Feature-Excel-Inspector-Screenshot-Command.md`
