In MakePlas.ps1 is it possible to insert a reference in the end to the claude session that generated the plan so I easily can open that session again?

## Outcome

Added session ID tracking to MakePlan so each generated plan includes a reference to the Claude session that created it.

### Changes

**MakePlan.ps1**: Generate a UUID before launching Claude, pass it via `--session-id` flag, and include it in the firmware header as `SessionId`.

**Program.md**: Updated plan format to include `session:` in frontmatter. Always included, sourced from the `SessionId` header arg. Plans can be reopened with `claude --resume <session-id>`.

No tools or memory created.
