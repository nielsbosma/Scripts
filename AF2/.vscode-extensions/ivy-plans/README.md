# Ivy Plans VS Code Extension

## Deployment

This extension is deployed via symlink — the folder is symlinked into `~/.vscode/extensions/`. No packaging (`.vsix`) or reinstall is needed.

After editing files (e.g., `package.json`, `extension.js`), run **Developer: Reload Window** in VS Code to pick up the changes.

This applies to all extensions under `D:\Repos\_Personal\Scripts\AF2\.vscode-extensions\`.
